DOCTYPE
- All HTML files must start with a document type declaration: <!DOCTYPE html>.
- <!DOCTYPE> declaration represents the document type, helps browsers to display web pages correctly.
- Only appears once at the top of the html file.
- Its declaration is not case sensitive.

<ol>
- Defines an ordered list.
<ul>
- Defines an unordered list.
<dl>
- Defines a description list.

The Difference between Text Editor and Word Processor
- Text editor like Notepad is only for unformatted text files, while word processor like Microsoft Word has more advanced features such as formatting, grammar checks and insertion of images.

Client-Server Model
- It's a distributed application structure that partitions tasks or workloads between the providers of a resource or service, called servers, and service requesters called clients.
- When the client computer sends a request for data to the server through the internet, the server accepts the requested process and delivers the data packets requested back to the client. clients do not share any of their resources.
- For example: Email.

FTP
- File Transfer Protocol.
- It's a network protocol for transmitting files between computers over Transmission Control Protocol/Internet Protocol (TCP/IP) connections.
HTTP
- Hyper Text Transfer Protocol.
- It's a protocol for fetching resources such as HTML documents.
TCP/IP
- Transmission Control Protocol/Internet Protocol.
- It's a networking protocol that allows two computers to communicate.
